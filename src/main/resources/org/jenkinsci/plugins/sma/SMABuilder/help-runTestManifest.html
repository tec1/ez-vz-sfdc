Enter the Manifest XML filename which resides inside of your repository. This manifest allows for more fine-grained controll of which Apex test classes should be run for a particular class.

The Manifest XML should be formatted like:

<?xml version="1.0" ?>
<Config>
    <mappings>
        <class>HNDL_Account</class>
        <tests>TST_HNDL_Account</tests>
        <tests>TST_VAT</tests>
        <tests>…</tests>
    </mappings>
    <mappings>
        <class>…</class>
        <tests>…</tests>
    </mappings>
</Config>
